Los frameworks de JavaScript son conjuntos de herramientas y librerías que facilitan el desarrollo de aplicaciones web. Estos frameworks ofrecen diversas funcionalidades y ayudan a los desarrolladores a ahorrar tiempo y esfuerzo al escribir código.

Existen muchos frameworks de JavaScript disponibles en la actualidad, pero los más populares y utilizados son:

![react-icon.png](../../public/images/react-icon.png)

## [React.js](https://es.reactjs.org/)

Desarrollado y mantenido por Facebook / Meta. Se utiliza principalmente para crear interfaces de usuario (UI) y se basa en el concepto de componentes reutilizables.

```jsx
// Código de ejemplo:
import React from 'react';
import ReactDOM from 'react-dom';

const App = () => {
  return (
    <div>
      <h1>Hola mundo!</h1>
      <p>Bienvenido a React.</p>
    </div>
  );
}

ReactDOM.render(<App />, document.getElementById('root'));

```

## Angular

Desarrollado y mantenido por Google. Ofrece una serie de herramientas y librerías para facilitar el desarrollo de aplicaciones web y móviles.

```jsx
import { Component } from '@angular/core';

@Component({
  selector: 'app-root',
  template: `
    <h1>Hola mundo!</h1>
    <p>Bienvenido a Angular.</p>
  `
})
export class AppComponent { }

```

## Vue

Es un framework de JavaScript open source y de código abierto. Se caracteriza por ser fácil de aprender y utilizar, y por su enfoque en la creación de interfaces de usuario.
